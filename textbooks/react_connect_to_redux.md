* Приложение начинается с редюсеров
* Редюсер - это чистая функция, которая принимает на вход состояние и экшн с данными (или без) и возвращает новое состояние приложения
* После этого создаётся store с помощью функции createStore, куда передаются наши редюсеры (их можно объединить с помощью combineRedusers)
* Чтобы подключить реакт к редаксу, наше приложение оборачивается в компонент Provider из библиотеки react-redux, куда передаётся props store
* К компонентам редакс подключается с помощью декоратора connect из библиотеки react-redux, который оборачивает наш компонент, куда первым аргументом передаётся mapStateToProps - функция, которая подписывает компонент на обновления store, принимает первым аргументом текущее состояние store и возвращает объект с данными, которые попадют в props. Вторым аргументом connect принимает объект с функциями (ActionCreators), каждая из которых попадает в props компонента.
